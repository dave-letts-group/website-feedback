Welcome to our platform

This demo page showcases the WebFeedback widget in action. The widget appears as a small floating button that lets visitors send feedback, report bugs, or request features — all with an automatic screenshot of the current page.

💡 Try it out: Click the purple chat button in the bottom-right corner. You can also change the position by editing the position attribute on the widget element.

📸 Screenshots

The widget automatically captures a screenshot of the page when the feedback dialog opens.

🏷️ Categories

Users can classify their feedback as a bug report, feature request, or general comment.

⭐ Ratings

Optional star rating to quickly gauge user sentiment about the page.

📋 Metadata

Attach arbitrary JSON data to each submission — user IDs, session info, feature flags, etc.

How it works

The widget is a standard Web Component. Add the script tag and the custom element to any HTML page. It uses Shadow DOM so its styles never conflict with your site.

Feedback is sent to your backend via a simple REST API, authenticated with a per-tenant site key. Admins can review all submissions in the dashboard at /admin.
